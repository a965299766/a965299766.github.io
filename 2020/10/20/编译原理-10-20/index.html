##1. 求First集和Follow集

例1
有如下文法：
① E -> TE’
② E’ -> +TE’ | ε
③ T -> FT’
④ T’ -> *FT’ | ε
⑤ F -> ( E ) | id
求各个文法符号的First集和Follow集。

1）First集
First(E) = First(T) = First(F) = {‘(‘, id}
First(E’) = {+, ε}
First(T’) = {*, ε}

2）Follow集
计算所有非终结符号A的Follow(A)时，不断应用如下规则，直到再没有新的终结符号可以被加入到任意的Follow集合中为止。
1. 将$放到Follow(S)中，其中S是开始符号，$是输入右端的结束标记；
即Follow(S)包含{$}，其中，S是开始符号。
2. 如果存在一个产生式A -> αBβ，那么First(β)中除ε之外的所有符号都在Follow(B)中；
即Follow(B)包含First(β) - {ε}。
3. 如果存在一个产生式 A -> αB，或存在产生式 A -> αBβ且FIRST(β)包含ε，那么Follow(A)中的所有符号都在Follow(B)中。
即Follow(B)包含Follow(A)。

2.0）由于E是起始符号，所以Follow(E)包含{$}；
2.1）求Follow(E)，看E出现在哪些产生式的右侧，发现出现在⑤式的右侧，观察发现此处E的下一个符号为终结符’)’，得出Follow(E)包含First(‘ ) ‘)，直接得出Follow(E)包含{‘ ) ‘}；再次观察发现E没有出现在其他式子的右侧，可以得出Follow(E) = {‘ ) ‘, $};
2.2）求Follow(E’)，E‘出现在①式和②式的右侧，先看①式，E’后面没有下一个符号了，可以得出Follow(E’)包含Follow(E)，即Follow(E’)包含{‘ ) ‘, $}；再看②式，E‘后面也没有下一个符号，则Follow(E’)包含Follow(E’)，这是废话，不管。此时没有其他式子右侧有E‘了，则Follow(E’) = {‘ ) ‘, $}；
2.3）求Follow(T)，观察T出现在哪些式子的右侧，发现出现在①式和②式的右侧，先看①式，T后面跟着E‘，可得Follow(T)包含First(E’) - {ε}，即Follow(T)包含{+}；再观察②式，发现E’还能推导出ε，那我们把产生式右侧的E‘用ε替代，产生式变为E’ -> +T，可知Follow(T)包含Follow(E’)，即Follow(T)包含{‘ ) ‘, $}。此外没有别的T出现在产生式右侧了，此时可以直接Follow(T) = {‘ ) ‘, $’} + First(E’) - {ε}，即Follow(T) = {‘ ) ‘, +, $}；
2.4）求Follow(T’)，看哪些式子右侧包含T’，发现③式和④式右侧有T‘，先看③，T’后面不跟符号了，则Follow(T’)包含Follow(T)，即Follow(T’)包含{‘ ) ‘, +, $}；再看④，可知Follow(T’)包含Follow(T‘)，废话，不管；得出Follow(T’) = Follow(T) = {‘ ) ‘, +, $}；
2.5）求Follow(F)，发现③式，④式右侧包含F，先看③，F后面跟T’，则Follow(F)包含First(T’) - {ε}，即Follow(F)包含{*}；再看④式，F后面跟着T‘，和上面一样处理；但是由④式还能得出，T’能推导出ε，此时产生式变为T‘ -> *F，可得Follow(F)包含Follow(T’)，即Follow(F)包含{‘ ) ‘, +, $}；此时没有别的产生式右侧有F了，即Follow(F) = {‘ ) ‘, + , *, $}。
2.6）结果：Follow(E) = {‘ ) ‘, $}；
Follow(E’) = {‘ ) ‘, $}；
Follow(T) = {‘ ) ‘, +, $}；
Follow(T’) = {‘ ) ‘, +, $}；
Follow(F) = {‘ ) ‘, +, *, $}。

例2
有如下文法：
① S -> iEtSS’ | a
② S’ -> eS | ε
③ E -> b
求各个文法符号的First集和Follow集。

1）First集
First(S) = {i, a}
First(S’) = {e, ε}
First(E) = {b}

2）Follow集
计算所有非终结符号A的Follow(A)时，不断应用如下规则，直到再没有新的终结符号可以被加入到任意的Follow集合中为止。
1. 将$放到Follow(S)中，其中S是开始符号，$是输入右端的结束标记；
即Follow(S)包含{$}，其中，S是开始符号。
2. 如果存在一个产生式A -> αBβ，那么First(β)中除ε之外的所有符号都在Follow(B)中；
即Follow(B)包含First(β) - {ε}。
3. 如果存在一个产生式 A -> αB，或存在产生式 A -> αBβ且FIRST(β)包含ε，那么Follow(A)中的所有符号都在Follow(B)中。
即Follow(B)包含Follow(A)。

2.0）由于S是起始符号，所以Follow(S)包含{$}；
2.1）求Follow(S)，观察哪些式子右侧有S，发现①式和②式右侧有S。先看①，S的后面跟着S’，可以得出Follow(S)包含First(S’) - {ε}，即Follow(S)包含{e}；但此时通过②式可以得出S‘能推导出ε，此时①式可以改写为S -> iEtS | a，此时得出Follow(S)包含Follow(S)，废话，不管；再看②式，S的后面不跟符号了，则得出Follow(S)包含Follow(S’)；此时Follow(S) = Follow(S’) + {e, $}；
2.2）求Follow(S’)，观察得知只有①式右侧有S’，而且S‘后面不跟符号，得出Follow(S’)包含Follow(S)，再由2.1）得出的结论Follow(S)包含Follow(S’)，得出Follow(S) = Follow(S’)，所以Follow(S) = Follow(S’) = {e, $}；
2.3）求Follow(E)， E只在①式右侧出现，且后面跟的是终结符t，则Follow(E) = First(t) = {t}；
2.6）结果：Follow(S) = {e, $}；
Follow(S’) = {e, $}；
Follow(E) = {t}；
