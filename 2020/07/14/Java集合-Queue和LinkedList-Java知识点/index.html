Java集合-Queue和LinkedList[Java知识点]

发表于 2020-07-14 | 分类于 Java知识点

Queue接口是我们平常所说的队列（数据先进先出），LinkedList是他的一个实现类
如果我们想定义一个队列，以下两种方法都是可以的：

Queue<E> queue=new LinkedList<E>();
LinkedList<E> queue=new LinkedList<E>();

但必须注意：如果把他定义为Queue，那它就是一个纯粹的队列；
而如果定义成LinkedList，它除了实现Queue的方法外，还实现了List的许多方法，我们既可以把他当队列，还可以当做双向队列、链表、栈来使用他。

##2. Queue的常用方法

boolean add(E e)

在队尾中增加新数据

boolean offer(E e)

在队尾增加新数据
与add()不同的是：当队列容量已满时，add()方法会抛出IllegalStateException异常，
而offer()方法会返回 false

E element()

获取队列头元素

E peek()

获取队列头元素
与element()不同的是：在队列元素为空的情况下，element() 方法会抛出NoSuchElementException异常，
而peek() 方法会返回 null

E poll()

删除队列头元素

boolean contains(Object o)

判断队列中是否存在元素o
存在返回true，否则返回false

boolean isEmpty()

判断队列是否为空
为空则返回true，否则返回false

int size()

获取队列的长度
即队列中元素的个数

boolean remove(Object o)
boolean remove(int index)

①. 如果定义的是Queue类型，则删除队列中第一个遇到的值等于o的元素；
②. 如果定义的是LinkedList类型，则删除队列中下标为index的元素。

void clear()

清空队列

##4. 除了使用Queue定义的方法之外，LinkedList还能像链表和栈一样操作数据，常用方法如下

void addFirst(E e)

从集合的头部插入元素

void addLast(E e)

从集合的尾部插入元素

E get(int index)

获取集合中给定下标处的值

E getFirst()

获取集合头部的值

E getLast()

获取集合尾部的值

int indexOf(Object o)

获取给定元素在集合中首次出现的下标

int lastIndexOf(Object o)

获取给定元素在集合中最后一次出现的下标

void push(E e)

往集合头部插入数据，像栈一样使用

E pop()

返回集合头部的数据，并删除
像栈一样使用
